Notes
====================

Purpose
--------------------

Notes is for writing down and saving all the thoughts you don't want to lose.

Tools used
--------------------

- Backend: Node.js, Express.js, Mongoose, Redis
- Database: MongoDB
- Frontend: React, Redux

How to run app locally
--------------------

- Frontend
  * `cd app/`
  * `nvm use` to use proper Node version
  * `yarn`
  * `yarn start`

Language
